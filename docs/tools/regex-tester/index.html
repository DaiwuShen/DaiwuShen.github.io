常用正则表达式符号对照表
符号 含义
^匹配开头
$匹配结尾
.匹配任意字符（换行符和回车符除外）
?匹配零次或一次，或限制贪婪
*匹配零次或多次
+匹配一次或多次
[a-z0-9]中括号表达式，匹配中括号内的包含的任一字符一次
[^a]中括号表达式，不匹配字符 a
\w匹配字符、数字、下划线一次，与[0-9a-zA-Z_]含义相同
\W匹配除了 \w 外的字符
()包裹子表达式
{n,m}匹配次数限定，最小匹配 n 次，最大匹配 m 次
